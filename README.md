# Dev Setup

Personal scripts and configuration files for bootstrapping a Debian-based Linux machine from scratch.

## About

This project automates the repetitive parts of setting up a development environment, going from a fresh OS install to a fully working machine in minutes instead of hours. Scripts are designed to run unattended using a simple configuration file for any values that change between installations.

Each script is tested and validated in a Docker container before running on real hardware.

## Planned Features

- **SSH & GPG** — Key generation and management
- **Git** — Configuration and credentials setup
- **Shell** — Customization and dotfiles deployment
- **Dev Environment** — Docker, Node.js, Python, and more
- **Tools & Utilities** — Common developer tools installation
- **Network** — Remote access and firewall configuration

## Status

Work in progress. Scripts are being developed and tested incrementally.
